# intuition-machine

带有旧纸质感（aged paper）和双语标注的技术简报信息图风格

## 设计美学

学术/技术简报风格，配有清晰的 2D 或等轴测技术插图。信息密集但组织有序，视觉层次清晰。复古蓝图（blueprint）美学与现代清晰度的结合。多解释性元素与双语标注。

## 背景

- 颜色：旧奶油色（#F5F0E6）
- 纹理：细微纸张纹理带浅折痕，复古技术印刷感

## 视觉元素

- 等轴测 3D 或扁平 2D 技术图表
- 带标注内容的解释性文本框
- 双语标注（英文 + 中文）
- 褪色主题背景图案
- 元素上的清晰黑色轮廓线
- 分栏或三联画（triptych）布局
- 关键洞察框

## 风格规则

### 宜

- 包含多个带内容的文本框
- 为关键元素使用双语标注
- 添加褪色主题背景图案
- 保持旧纸张质感
- 创建清晰的视觉层次

### 忌

- 创建写实 3D 渲染
- 让插图没有解释性文本
- 在角落添加印章或水印
- 使用渐变或光面效果
- 使其看起来过于现代/数字化

## 适用于

技术解释、概念分解、学术内容、研究总结、双语受众、知识文档
