# 极简风格

极致简洁、精致高级美学。

## 元素组合

```yaml
canvas:
  ratio: portrait-3-4
  grid: single

image_effects:
  cutout: clean
  stroke: none | white-solid
  filter: none | muted-tones

typography:
  decorated: none
  tags: white-black | pill
  direction: horizontal

decorations:
  emphasis: underline | circle-mark
  background: solid-white | solid-pastel
  doodles: hand-drawn-lines
  frames: none | rounded-rect
```

## 调色板

| 角色 | 颜色 | Hex |
|------|--------|-----|
| 主色 | 黑色、白色 | #000000, #FFFFFF |
| 背景 | 米白色、纯白色 | #FAFAFA, #FFFFFF |
| 点缀色 | 单色（从内容提取） | 蓝色、绿色或珊瑚色 |

## 视觉元素

- 单一焦点、细线条
- 最大留白
- 简洁干净的装饰
- 克制的视觉元素

## 排版

- 干净简洁的手写体
- 最少的字重变化
- 优雅间距

## 最佳布局搭配

| 布局 | 兼容性 | 适用场景 |
|--------|---------------|----------|
| sparse | ✓✓ | 优雅金句 |

| dense | ✓✓ | 干净知识卡 |
