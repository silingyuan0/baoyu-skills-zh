# 学习笔记风格

真实的手写笔记照片美学 — 学生笔记风格，密集但可读。

## 元素组合

```yaml
canvas:
  ratio: portrait-3-4
  grid: single

image_effects:
  cutout: none
  stroke: none
  filter: natural-photo

typography:
  decorated: none
  tags: none
  direction: horizontal

decorations:
  emphasis: circle-mark | underline | checkmark | cross | star-simple
  background: lined-paper-white
  doodles: arrows-simple | margin-notes | corrections | explanatory-diagrams
  frames: none
```

## 配色（三色标注系统）

| 作用 | 颜色 | Hex |
|------|--------|-----|
| 主色 | 蓝色圆珠笔、黑墨水 | #1E3A5F, #1A1A1A |
| 高亮 | 黄色荧光笔 | #FFFF00（50% 透明度）|
| 强调 | 红色圆珠笔（圈、划线）| #CC0000 |
| 背景 | 白色横线纸 | #FFFFFF |

## 视觉元素

- 真实照片视角：俯视书桌
- 手持蓝色圆珠笔的手，积极地做着标注
- 极其密集的手写内容，充满整页
- 红色笔批注：圈、划线、星星、方框
- 黄色荧光笔标记关键术语
- 批改标记、挤进页边的潦草笔记
- 简单的手绘符号：→ * ✓ ✗ !
- 不同的笔压力创造深浅笔画

## 字体

- 真实的学生手写字
- 潦草但可读，保持清晰结构
- 不同字体大小（大标题、小正文、微小页边注释）
- CJK 优化

## 内容结构

三段式布局：

### 顶部区域
- 核心主题（红色圈了多圈）
- 第一个章节标题 + 3-4 个关键点
- 箭头连接、红色下划线

### 中间区域
- 第二个章节标题（红色笔框）
- 编号步骤 ①②③
- 具体方法和补充笔记

### 底部区域
- 第三个章节标题（红色星星）
- 时间点/关键指标
- 关键引言/核心技巧（微小角注）

## 最佳布局搭配

| 布局 | 兼容性 | 适用于 |
|--------|---------------|----------|
| sparse | 不适合 | 风格需要密集内容 |
| balanced | 适合 | 内容较轻时 |
| dense | 非常适合 | 知识笔记、总结 |
| list | 非常适合 | 步骤清单、排名 |
| comparison | 适合 | 比较分析 |
| flow | 适合 | 流程 |
| mindmap | 非常适合 | 思维导图笔记 |
| quadrant | 适合 | 四象限分析 |

## 适用于

- 学习指南、考试笔记
- 知识整理、框架总结
- 教程总结、快速笔记
- "学霸笔记"风格内容
- 需要真实感的知识分享

## 风格规则

### 做
- 保持内容极其密集
- 使用简单符号（→ * ✓ ✗ !）
- 用红色笔批注关键点
- 包含批改标记
- 把小笔记挤进页边

### 不做
- 使用复杂表情包
- 留太多空白
- 做整洁、整齐的布局
- 添加彩色装饰
- 包含卡通元素
