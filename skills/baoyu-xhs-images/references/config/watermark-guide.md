# 水印指南

## 位置示意图

```
┌─────────────────────────────┐
│                  [右上角]    │
│                             │
│                             │
│         图像内容             │
│                             │
│                             │
│[左下角][底部居中][右下角]    │
└─────────────────────────────┘
```

## 位置推荐

| 位置 | 适用于 | 避免用于 |
|----------|----------|------------|
| `bottom-right` | 默认选择，最常见 | 右下角有重要信息 |
| `bottom-left` | 右侧内容较多的布局 | 左下角有重要信息 |
| `bottom-center` | 居中设计 | 底部文字较多 |
| `top-right` | 底部内容较多 | 右上角有标题/标题栏 |

## 内容格式

| 格式 | 示例 | 风格 |
|--------|---------|-------|
| Handle | `@username` | 小红书最常见 |
| 文本 | `MyBrand` | 简单品牌标识 |
| 中文 | `小红书:用户名` | 平台特定 |
| URL | `myblog.com` | 跨平台 |

## 最佳实践

1. **一致性**：系列所有图像使用相同水印
2. **可读性**：确保水印在浅色/深色区域都清晰可读
3. **大小**：保持低调 - 不应干扰内容

## 提示词集成

启用水印时，添加到图像生成提示词：

```
Include a subtle watermark "[content]" positioned at [position].
The watermark should be legible but not distracting from the main content.
```

## 常见问题

| 问题 | 解决方案 |
|-------|----------|
| 水印不可见 | 调整位置或检查对比度 |
| 水印过于突出 | 更换位置或减小大小 |
| 水印遮挡内容 | 更换位置 |
| 系列图像不一致 | 使用会话 ID 确保一致性 |
