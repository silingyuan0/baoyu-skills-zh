# 小红书内容分析框架

专为小红书独特参与模式量身定制的深度分析框架。

## 目的

在创建信息图之前，深入分析源材料以：
- 最大化钩子力量和滑动动机
- 识别值得收藏和值得分享的元素
- 规划视觉叙事弧线
- 将内容匹配到最佳风格/布局

## 平台特性

与其他平台不同，小红书内容必须优先考虑：
- **钩子力量**：首图决定 90% 的参与度
- **滑动动机**：每张图必须促使用户继续滑动
- **收藏价值**：值得收藏以备后用的内容
- **分享触发**：推动分享的情感共鸣

## 分析维度

### 1. 内容类型分类

| 类型 | 特点 | 最佳风格 | 最佳布局 |
|------|----------------|-------------|-------------|
| 种草/安利 | 产品推荐、注重利益 | cute/fresh | balanced/list |
| 干货分享 | 知识、技巧、教程 | notion | dense/list |
| 个人故事 | 个人经历、情感 | warm | balanced |
| 测评对比 | 评测、对比、优缺点 | bold/notion | comparison |
| 教程步骤 | 步骤指南 | fresh/notion | flow/list |
| 避坑指南 | 警告、需要避免的错误 | bold | list/comparison |
| 清单合集 | 集合、推荐 | cute/minimal | list/dense |

### 2. 爆款标题分析

使用以下模式评估标题/钩子潜力：

**钩子类型**：
- **数字钩子**："5个方法"、"3分钟学会"、"99%的人不知道"
- **痛点钩子**："踩过的坑"、"后悔没早知道"、"别再..."
- **好奇钩子**："原来..."、"竟然..."、"没想到..."
- **利益钩子**："省钱"、"变美"、"效率翻倍"
- **身份钩子**："打工人必看"、"学生党"、"新手妈妈"

**评分标准**：
- 五星（5/5）：多个强钩子组合
- 四星（4/5）：清晰钩子，有增强空间
- 三星（3/5）：基本钩子，需要加强
- 二星（2/5）：弱钩子，需要重大改进
- 一星（1/5）：无清晰钩子

### 3. 用户画像

| 受众 | 兴趣 | 偏好风格 | 内容重点 |
|----------|-----------|-----------------|---------------|
| 学生党 | 省钱、学习、校园 | cute/fresh | 平价、教程、学习方法 |
| 打工人 | 效率、职场、减压 | minimal/notion | 工具、技巧、摸鱼 |
| 宝妈 | 育儿、家居、省心 | warm/fresh | 实用、安全、经验 |
| 精致女孩 | 美妆、穿搭、仪式感 | cute/retro | 好看、氛围、品质 |
| 技术宅 | 工具、效率、极客 | notion/chalkboard | 深度、专业、新奇 |
| 美食爱好者 | 探店、食谱、测评 | warm/pop | 好吃、简单、颜值 |
| 旅行达人 | 攻略、打卡、小众 | fresh/retro | 省钱、避坑、拍照 |

### 4. 互动潜力

**收藏价值**：
- 是参考资料吗？高收藏潜力
- 是清单或列表吗？高收藏潜力
- 是教程吗？高收藏潜力
- 是时效性新闻吗？低收藏潜力

**分享触发**：
- "我朋友也需要看这个" → 高分享潜力
- "这说的就是我" → 身份共鸣
- "太有用了必须分享" → 实用分享
- "笑死，给朋友看看" → 娱乐分享

**评论诱导**：
- 开放式问题："你是哪种类型？"
- 经验分享："评论区说说你的经历"
- 辩论触发："你觉得呢？"
- 帮助请求："有更好的推荐吗？"

**互动设计**：
- 投票："A还是B？"
- 挑战："你能做到几个？"
- 标签："@你那个需要的朋友"

### 5. 视觉机会地图

| 内容元素 | 视觉处理 | 示例 |
|------------------|------------------|---------|
| 数据/统计 | 高亮数字、简单图表 | "节省80%时间" 大字突出 |
| 对比 | 前后对比、左右并排 | 左右分屏对比图 |
| 步骤 | 编号流程、箭头 | 1→2→3 流程图 |
| 清单 | 带图标的清单 | ✓/✗ 列表配图标 |
| 情感 | 人物表情、场景 | 卡通人物表情包 |
| 产品 | 产品展示、生活方式 | 产品实拍+使用场景 |
| 引用 | 引言卡片、对话气泡 | 金句卡片设计 |

### 6. 滑动流程设计

规划跨图像的叙事弧线：

| 位置 | 目的 | 钩子策略 |
|----------|---------|---------------|
| **封面** | 停止滑动 | 最强视觉冲击 + 核心标题 |
| **铺垫** | 建立语境 | 痛点共鸣 / 好奇心 |
| **核心** | 传递价值 | 干货内容，每页1-2个要点 |
| **收获** | 实际收获 | 可执行的行动建议 |
| **结尾** | 推动行动 | CTA + 互动引导 |

**图像间的滑动动机**：
- 每张图以下一个的钩子结束
- 使用"下一页更精彩"类型过渡
- 创造需要滑动才能填补的信息缺口
- 通过编号建立期待（"第3个最重要"）

## 输出格式

分析结果应保存到 `analysis.md`：

```yaml
---
title: "5个让你效率翻倍的AI工具"
topic: 干货分享
content_type: 工具推荐
source_language: zh
user_language: zh
recommended_image_count: 6
---

## 用户画像

- **主要**：打工人、自由职业者 - 追求效率提升
- **次要**：学生党 - 写论文、做作业需要
- **补充**：内容创作者 - 需要AI辅助

## 钩子分析

**标题钩子评分**：四星（4/5）
- ✓ 数字钩子："5个"
- ✓ 利益钩子："效率翻倍"
- △ 可增强：加入身份标签 "打工人必看"

**建议优化**：
- 原标题："5个让你效率翻倍的AI工具"
- 优化："打工人必看！5个让我效率翻倍的AI神器"

## 价值主张

**为什么用户要看？**
1. **实用价值**：直接可用的工具推荐
2. **省时省力**：不用自己筛选，直接抄作业
3. **FOMO**：别人都在用，我不能落后

**收藏理由**：工具清单，需要时可以回来查

## 互动设计

- **互动点**：结尾问"你最常用哪个？"
- **评论诱导**："还有什么好用的工具评论区分享"
- **分享触发**：打工人会转发给同事

## 内容信号

- "AI工具" → notion + dense
- "效率" → notion + list
- "干货" → minimal + dense

## 滑动流程

| 图像 | 位置 | 目的 | 钩子 |
|-------|----------|---------|------|
| 1 | 封面 | 吸引停留 | 标题+视觉冲击 |
| 2 | 铺垫 | 建立共鸣 | 为什么需要AI工具 |
| 3-5 | 核心 | 核心价值 | 每页1-2个工具详解 |
| 6 | 结尾 | 行动引导 | 总结+互动引导 |

## 推荐方案

1. **Notion + Dense** - 知识卡片风格，适合干货分享（推荐）
2. **Notion + List** - 清爽知识卡片风格
3. **Minimal + Balanced** - 简约高端，适合职场人群
```

## 分析清单

在进入大纲生成之前：

- [ ] 我能识别内容类型吗？
- [ ] 钩子足够强吗？（≥3星）
- [ ] 我知道主要受众是谁吗？
- [ ] 我已识别收藏/分享触发器了吗？
- [ ] 有清晰的视觉机会吗？
- [ ] 滑动流程已规划了吗？
- [ ] 我已确定最佳风格+布局推荐了吗？
