## 封面设计原则

- 学术庄重感与视觉吸引力
- 反映知识/科学主题的标题排版
- 暗示核心主题的构图（角色剪影、标志性符号、概念图）
- 用于史诗范围的副标题或时间跨度

## 面板构图指南

| 面板类型 | 推荐数量 | 用途 |
|-----------|-------------------|-------|
| 主叙事 | 每页 3-5 | 故事推进 |
| 概念图 | 每页 1-2 | 可视化抽象概念 |
| 叙述者面板 | 每页 0-1 | 评论、过渡 |
| Splash（全幅/半幅） | 偶尔 | 重要时刻 |

## 面板尺寸参考

- **全页（Splash）**：重要时刻、关键突破
- **半页**：重要场景、转折点
- **1/3 页**：标准叙事面板
- **1/4 或更小**：快速推进、顺序动作

## 概念可视化技术

将抽象概念转化为具体视觉：

| 抽象概念 | 视觉方法 |
|-----------------|----------------|
| 神经网络 | 带连接线的发光节点 |
| 梯度下降 | 球在山谷地形上滚动 |
| 数据流 | 发光粒子在管道中流动 |
| 算法迭代 | 上升螺旋楼梯 |
| 突破时刻 | 破碎的屏障、穿透的光 |
| 逻辑证明 | 正在组装的构建块 |
| 不确定性 | 分叉路径、雾、多个影子 |

## 文本元素设计

| 文本类型 | 风格 | 用途 |
|-----------|-------|-------|
| 角色对话 | 椭圆形对话气泡 | 主要叙事对话 |
| 叙述者评论 | 矩形框 | 解释、评论 |
| 标题栏 | 边缘安装的矩形 | 时间、地点信息 |
| 思考气泡 | 云形状 | 角色内心独白 |
| 术语标签 | 粗体 / 特殊颜色 | 技术术语首次出现 |

## 用于一致性的提示词结构

每个页面提示词应包含角色参考：

```
[角色参考]
（来自 characters.md 的关键细节，用于本页面中的角色）

[页面内容]
（特定场景、面板布局和视觉元素）

[一致性提醒]
保持角色参考中定义的确切角色外观。
- [角色 A]：[关键识别特征]
- [角色 B]：[关键识别特征]
```
