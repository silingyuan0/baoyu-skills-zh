# dramatic

戏剧基调 - 高对比、强烈、震撼时刻

## 概述

高冲击力的戏剧性色调，用于关键时刻、冲突和突破。使用强烈的对比和紧张的构图创造情感力量。

## 情感特征

- 张力与强烈感
- 关键时刻
- 冲突与化解
- 突破性发现
- 情感高潮

## 颜色修正

应用于任何画风时：

| 调整项 | 方向 |
|--------|------|
| 饱和度 | 高（鲜艳或深沉） |
| 对比度 | 最大 |
| 色温 | 根据效果变化 |
| 亮度 | 强高光、深阴影 |

## 对比度处理

- 明暗分界清晰锐利
- 极少中间色调
- 强烈的构图
- 剪影效果
- 边缘光效果

## 强调色

- 深海军蓝（#1A365D）
- 深红（#9B2C2C）
- 纯白
- 浓重黑色
- 每场景限制调色板

## 光照

- 戏剧性单光源
- 高对比度阴影
- 人物边缘光
- 聚光灯效果
- 明暗对照法（Chiaroscuro）风格

## 情感范围

| 情感 | 表达方式 |
|------|----------|
| 愤怒 | 强烈、棱角分明的特征 |
| 决心 | 坚定、专注的目光 |
| 震惊 | 睁大眼睛、锐利光线 |
| 胜利 | 强有力的、高昂的姿态 |

## 构图

- 棱角分明的动态布局
- 戏剧性的摄影角度
- 仰视/俯视视角
- 对角线构图
- 负空间增强冲击力

## 视觉元素

- 速度线营造张力
- 冲击效果
- 戏剧性背景（暴风雨、火焰）
- 剪影
- 光芒爆发效果
- 环境戏剧效果

## 适合场景

- 关键性发现
- 冲突场景
- 高潮时刻
- 突破性领悟
- 情感对峙
- 历史转折点

## 搭配建议

特别适合与以下画风搭配：
- realistic：强烈的戏剧效果
- ink-brush：武侠高潮
- ligne-claire：历史转折
- manga：少年战斗

避免搭配：chalk（风格不匹配）
