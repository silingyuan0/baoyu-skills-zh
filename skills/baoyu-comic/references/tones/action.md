# action

动作基调 - 速度、冲击、力量

## 概述

高冲击力的动作氛围，具有动态运动、战斗效果和强大的视觉能量。营造震撼人心的激烈战斗画面。

## 情感特征

- 速度与运动
- 力量与冲击
- 战斗强度
- 身体能量
- 肾上腺素飙升的刺激感

## 颜色修正

应用于任何画风时：

| 调整项 | 方向 |
|--------|------|
| 饱和度 | 高对比 |
| 对比度 | 最大 |
| 色温 | 根据效果变化 |
| 亮度 | 动态范围 |

## 动作效果

**战斗/运动效果**（大量运用）：

| 效果 | 用途 |
|------|------|
| 速度线 | 运动、速度 |
| 冲击爆发 | 打击、碰撞 |
| 冲击波 | 强力冲击 |
| 飞散碎片 | 环境破坏 |
| 尘云 | 地面冲击 |
| 运动模糊 | 快速移动 |
| 残影 | 超高速 |

## 特殊效果

| 效果类型 | 视觉表现 |
|----------|----------|
| 能量攻击 | 发光、辐射 |
| 物理冲击 | 放射线、碎片 |
| 运动 | 速度线、模糊 |
| 氛围 | 飞舞的粒子、风 |

## 效果颜色

- 动态角度
- 极端透视
- 突破分格的布局

## 姿势指导

- 动态的战士姿势
- 可见的重量和动量

- 捕捉运动的流动感
- 强调冲击点

## 适合场景

- 武打格斗
- 动作场景
- 体育时刻
- 身体挑战
- 战斗场面
- 高潮对决

## 搭配建议

特别适合与以下画风搭配：
- ink-brush：武侠格斗
- manga：少年战斗

避免搭配：
- chalk：风格不匹配
- ligne-claire：风格不匹配（过于静态）
