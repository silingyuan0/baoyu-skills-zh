# neutral

中性基调 - 平衡、理性、教育导向

## 概述

默认的平衡色调，适用于教育类和信息性内容。不过分情感化也不过于冷淡 — 创建专业且平易近人的氛围。

## 情感特征

- 平衡的情感表达
- 清晰、理性的呈现方式
- 教育导向
- 专业但平易近人
- 客观的叙事方式

## 颜色修正

应用于任何画风时：

| 调整项 | 方向 |
|--------|------|
| 饱和度 | 标准（无偏移） |
| 对比度 | 平衡 |
| 色温 | 中性 |
| 亮度 | 略微偏亮 |

## 光照

- 均匀、清晰的照明
- 极少戏剧性阴影
- 各分格之间保持一致
- 自然光源
- 无极端对比

## 情感范围

| 情感 | 表达程度 |
|------|----------|
| 喜悦 | 适度微笑 |
| 担忧 | 沉思的表情 |
| 惊讶 | 轻微睁大眼睛 |
| 沮丧 | 微微皱眉 |

## 构图

- 平衡的分格布局
- 清晰的视觉焦点
- 可读的层次结构
- 标准取景
- 功能性构图

## 适合场景

- 教育类内容
- 技术教程
- 信息性传记
- 纪录片风格
- 专业题材

## 使用说明

中性是默认色调。可与任何画风搭配，作为基础专业输出。最通用的色调选项。
