# warm

温馨基调 - 怀旧、私密、温暖治愈

## 概述

温暖而亲切的氛围，适用于个人故事和怀旧内容。通过舒适的视觉美学和治愈系的画面创造情感连接。

## 情感特征

- 怀旧感
- 私密、亲近的氛围
- 温暖治愈
- 回忆与反思
- 柔和的情感温度

## 颜色修正

应用于任何画风时：

| 调整项 | 方向 |
|--------|------|
| 饱和度 | 略微降低 |
| 对比度 | 柔化 |
| 色温 | 暖色偏移（+15%） |
| 亮度 | 柔和、金黄 |

## 色温

将调色板偏移为暖色调：

| 原始色 | 暖色偏移 |
|--------|----------|
| 冷蓝 | 柔和青色 |
| 纯白 | 奶油色 |
| 灰色 | 暖灰色 |
| 黑色 | 柔和炭黑 |

## 强调色

- 金黄色（#D69E2E）
- 柔和橙色（#DD6B20）
- 暖棕色（#8B6F47）
- 日落色调

## 光照

- 黄金时刻光线
- 柔和、漫射光
- 温暖的室内灯光
- 烛光/台灯的温暖
- 柔和的阴影

## 情感范围

| 情感 | 表达方式 |
|------|----------|
| 喜悦 | 真诚温暖的微笑 |
| 悲伤 | 柔和的忧伤 |
| 爱 | 温柔、细腻的表情 |
| 回忆 | 悠远、沉思的目光 |

## 构图

- 亲近的取景
- 舒适的环境
- 柔焦背景
- 温馨的空间
- 突出个人时刻

## 视觉元素

- 暖色光线
- 柔和的边缘
- 怀旧道具（旧照片、纪念品）
- 治愈物件（毯子、茶杯）
- 自然元素（秋叶、日落）

## 适合场景

- 个人故事
- 童年回忆
- 导师叙事
- 家族历史
- 温和的传记
- 治愈之旅

## 搭配建议

特别适合与以下画风搭配：
- ligne-claire：怀旧欧洲漫画
- realistic：感人的人物故事
- manga：日常生活的温暖
- chalk：怀旧教育风格
