# shoujo

少女预设 - Classic shoujo manga with romantic aesthetics

## 基础配置

| 维度 | 值 |
|-----------|-------|
| 艺术风格 | manga |
| 基调 | romantic |
| 布局 | standard（默认）|

等同于：`--art manga --tone romantic`

## 独特规则

此预设包含超出 art+tone 组合的特殊规则。使用 `--style shoujo` 时，必须应用以下所有规则。

### 装饰元素（必需）

每个情感时刻必须包含装饰元素：

| 情感 | 必需装饰 |
|---------|---------------------|
| 爱 | 漂浮的心、闪光、玫瑰花瓣 |
| 渴望 | 羽毛、气泡、远方闪光 |
| 喜悦 | 绽放的花朵、光芒爆发、星星 |
| 悲伤 | 飘落的花瓣、褪色的闪光 |
| 害羞 | 柔和闪光、漂浮的气泡 |
| 顿悟 | 带闪光的辐射线 |

### 眼睛细节要求

眼睛在少女风格中至关重要：

| 方面 | 处理 |
|--------|-----------|
| 大小 | 比标准漫画大（1.2倍）|
| 高光 | 多个（3-5个），放置表达情感 |
| 反射 | 情感时刻的场景反射 |
| 闪光 | 内置闪光效果 |
| 眼泪 | 水晶般详细的泪珠 |

### 角色美丽标准

| 特征 | 处理 |
|---------|-----------|
| 头发 | 飘逸、详细发丝、光泽高光 |
| 皮肤 | 瓷器感、面颊柔和红晕 |
| 嘴唇 | 柔和、略光泽 |
| 手 | 优雅、表达手势 |
| 姿势 | 优雅、优美姿势 |

### 背景效果

**情感时刻的抽象背景**：

| 时刻类型 | 背景 |
|-------------|-----------|
| 爱之告白 | 柔和渐变 + 漂浮花朵 |
| 震惊 | 网纸速度线 + 闪光 |
| 记忆 | 梦幻模糊 + 散落花瓣 |
| 顿悟 | 辐射线 + 光芒爆发 |
| 亲密 | 柔焦 + 漂浮元素 |

### 面板流程

- 重叠面板用于亲密时刻
- 打破面板边框增强情感冲击
- 在面板之间漂浮装饰元素
- 使用网纸渐变营造氛围
- 不规则面板形状增强戏剧性

### 情感节拍计时

放慢节奏以增强情感冲击：

| 场景类型 | 面板处理 |
|------------|-----------------|
| 告白 | 多个小面板，然后是 splash page |
| 眼神接触 | 特写序列 |
| 触碰 | 慢动作面板分解 |
| 顿悟 | 积累面板然后冲击 |

### 调色板应用

| 场景类型 | 调色板 |
|------------|---------|
| 浪漫 | 粉色、薰衣草、玫瑰金 |
| 快乐 | 柔黄、桃色、天蓝 |
| 悲伤 | 淡蓝、银色、灰薰衣草 |
| 戏剧 | 深玫瑰、紫、对比 |

### 网纸使用

| 氛围 | 网纸图案 |
|------|-------------|
| 中性 | 干净、最少 |
| 浪漫 | 柔和渐变叠加 |
| 戏剧 | 重对比色调 |
| 梦幻 | 柔和点图案 |
