# periodic-table

具有一致单元格格式的分类元素网格。

## 结构

- 矩形网格布局
- 每个单元格代表一个元素
- 颜色编码的类别区分
- 统一的单元格格式
- 可选的分组间隙

## 最佳适用

- 分类集合展示
- 工具/资源目录
- 技能矩阵
- 元素集合
- 参考指南

## 视觉元素

- 统一的单元格尺寸
- 类别颜色区分
- 符号/缩写突出显示
- 每个单元格配有小图标
- 类别图例

## 文本布局

- 标题置于顶部
- 单元格内容：符号、名称、简要信息
- 图例中标明类别名称
- 可选的行列标题
- 特殊情况说明脚注

## 推荐搭配

- `pop-art`：充满活力的元素网格展示
- `pixel-art`：复古风格集合展示
- `corporate-memphis`：商业工具目录
