# winding-roadmap

弯曲路径展示带有里程碑和检查点的旅程。

## 结构

- S形曲线或蜿蜒路径
- 沿途的里程碑
- 起点和目的地
- 侧面元素（障碍物、助力元素）
- 进度指示器

## 最佳适用

- 项目路线图
- 职业发展路径
- 客户旅程
- 学习路径
- 战略时间线

## 视觉元素

- 弯曲的道路或河流
- 里程碑标记/旗帜
- 沿途的场景元素
- 旅程中的车辆/人物
- 目的地地标

## 文本布局

- 标题置于顶部
- 各点的里程碑标签
- 路径分段名称
- 目的地描述
- 可选的时间线指示器

## 推荐搭配

- `storybook-watercolor`：充满奇思妙想的旅程风格
- `cartoon-hand-drawn`：友好的路线图风格
- `isometric-3d`：技术项目路径
