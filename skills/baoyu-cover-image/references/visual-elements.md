# 视觉元素库

按主题组织的图标和符号词汇表。使用这些作为封面构图的基础元素。

## 科技与开发

| 元素 | 适用于 |
|---------|---------|
| Code window / Terminal | 编程、开发 |
| Gear / Cog | 工程、设置、流程 |
| Circuit board / Chip | 硬件、AI、计算 |
| Binary / Data stream | 数据、算法 |
| API brackets `</>` | Web开发、API |
| Cloud | 云计算、SaaS |
| Lock / Shield | 安全、隐私 |
| Network nodes | 分布式系统、连接 |

## 组合模式

| 组合 | 代表 |
|-------------|------------|
| Lightbulb + Gear | 创新工程 |

| Bridge + People | 团队连接 |
| Magnifier + Data | 分析、洞察 |

## 渲染特定处理方式

| 渲染方式 | 元素风格 |
|-----------|---------------|
| `flat-vector` | 几何、简单形状、统一填充 |
| `hand-drawn` | 草图感、有机、手绘风格 |
| `painterly` | 柔和边缘、笔触感 |
| `digital` | 精准、渐变提示、精致 |
| `pixel` | 8位像素、网格对齐 |
| `chalk` | 粉质感、纹理、黑板风格 |
