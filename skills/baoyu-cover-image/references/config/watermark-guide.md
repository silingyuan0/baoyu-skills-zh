# 水印指南

## 位置示意图

```
┌─────────────────────────────┐
│                  [top-right]│
│                             │
│                             │
│       COVER IMAGE           │
│                             │
│                             │
│[bottom-left][bottom-center][bottom-right]│
└─────────────────────────────┘
```

## 位置建议

| 位置 | 适用于 | 避免用于 |
|----------|----------|------------|
| `bottom-right` | 默认选择，最常见 | 标题在右下角 |
| `bottom-left` | 右侧重布局 | 左下角有重要视觉 |
| `bottom-center` | 居中设计 | 底部文字密集区域 |
| `top-right` | 底部内容为主 | 右上角有标题/页眉 |

## 内容格式

| 格式 | 示例 | 风格 |
|--------|---------|-------|
| 社交媒体账号 | `@username` | 社交媒体 |
| 域名 | `myblog.com` | 跨平台 |
| 品牌 | `MyBrand` | 简单品牌 |
| 中文 | `博客名` | 中文平台 |

## 最佳实践

1. **一致性**：所有封面使用相同水印
2. **可读性**：确保水印在浅色/深色区域都可读
3. **大小**：保持低调 - 不应分散内容注意力

## 提示词集成

启用水印时，添加到图像生成提示词：

```
Include a subtle watermark "[content]" positioned at [position].
The watermark should be legible but not distracting from the main content.
```

## 封面特定注意事项

| 宽高比 | 推荐位置 | 说明 |
|--------------|---------------------|-------|
| 2.35:1 | bottom-right | 电影感 - 保持角落干净 |
| 16:9 | bottom-right | 标准 - 位置灵活 |
| 1:1 | bottom-center | 方形 - 居中通常效果更好 |

## 常见问题

| 问题 | 解决方案 |
|-------|----------|
| 水印不可见 | 调整位置或检查对比度 |
| 水印太突出 | 更换位置或减小尺寸 |
| 水印与标题重叠 | 更换位置或减小标题区域 |
